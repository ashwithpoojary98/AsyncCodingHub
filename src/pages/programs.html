Java Main Method The main method in Java is the entry point of any standalone Java application. It is where the program begins its execution. Without a properly defined main method, the Java Virtual Machine (JVM) cannot run the program. Signature of the Main Method Java 
              public static void main(String[] args)
Example 
        public class HelloWorld {
                public static void main(String[] args){
                    System.out.println("Hello World")
                }
            }
Output: 
    Hello World
